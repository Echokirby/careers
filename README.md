# AI-Native Super Analyst · 全栈研究员

> 一个以持续进化为底色的对冲基金，正在寻找少数真正卓越的人。

---

## 我们是谁

我们是一家对冲基金，正经历一场由AI驱动的组织变革——不是在现有框架里加几个AI工具，而是从根本上重构投研的生产方式。

我们相信，下一代顶级研究机构将由极少数的「超级个体」构成：每一个人都能借助AI，完成过去需要整支团队才能完成的工作。我们正在构建这样的组织，并寻找与我们共同走这条路的人。

---

## 角色定位

这不是一个单一职位的招聘。我们寻找的是能在以下三个方向中至少一个方向真正发力的人：

### 方向一 · 数据资产专家
将杂乱的现实世界转化为有价值的数据资产。利用AI处理非结构化信息——卫星图像、社交媒体、产业链调研、另类数据——并将其系统化为可复用的投研基础设施。

### 方向二 · 技能构建者
用 Agentic Workflow 重构研究链路。你理解 LLM 的能力边界，能设计并维护复杂的多智能体系统，让研究员从重复性工作中解放出来，专注于真正需要判断力的环节。

### 方向三 · Alpha 挖掘者
AI 武装的研究员。你有金融直觉，也有技术能力，能独立完成从信息获取、数据清洗、模型构建到投资决策的全链路。你不依赖团队支持，你就是那个团队。

## 我们在寻找什么样的人

**必要条件**

- 渴望成功，追求卓越——技术背景嫁接资本市场，或金融背景被 AI 重新武装
- 拥抱变化，不惧挑战，享受高频迭代带来的成就感
- 极强的好奇心与悟性，具备连点成线的迁移学习能力
- 极致工具主义者，站在前沿工具链实践第一线

**加分项**

- 自主构建过复杂 Agent 工作流，或维护过有实际用户的硬核开源项目
- 顶级买方/卖方/科技 VC/AI Lab 的实战经历
- 职业起步期即展现出远超同侪的学习速度
